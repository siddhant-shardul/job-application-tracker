\# Student Placement / Job Application Tracker



A full-stack project for tracking student placement and job applications.



\## Tech Stack



\- Backend: Java, Spring Boot

\- Frontend: React

\- Database: PostgreSQL

\- Version Control: Git/GitHub

\- API Testing: Postman

\- Deployment: Render/Railway first, AWS later



\## Current Status



\- Spring Boot backend created

\- PostgreSQL database connected

\- Git initialized



\## Planned Features



\- Student registration and login

\- Admin login

\- JWT authentication

\- Student profile with skills

\- Company management

\- Job application tracking

\- Resume upload

\- Dashboard

\- Admin panel
